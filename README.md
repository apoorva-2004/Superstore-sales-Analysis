##  Superstore Sales Analysis — Excel | Power BI | SQL | Python


###  **Detailed Project Overview**

This project demonstrates a complete **data analytics lifecycle** — from raw data exploration to actionable insights — using **four key tools**:

* **Excel** for initial data cleaning and basic analysis
* **SQL** for querying and data manipulation
* **Power BI** for interactive dashboards
* **Python (Pandas, Matplotlib, Seaborn)** for deep exploratory analysis and visualization

The dataset used is **Sample Superstore**, containing information on orders, sales, profits, and regions.

###  **Objectives**

* Clean, explore, and visualize the Superstore dataset
* Identify top-performing categories, sub-categories, and regions
* Analyze the relationship between discount and profit
* Track sales performance over time
* Build a professional business dashboard


###  **Tech Stack**

| Tool         | Purpose                                              |
| ------------ | ---------------------------------------------------- |
| **Excel**    | Data cleaning, pivot tables, trend analysis          |
| **SQL**      | Data extraction, filtering, and aggregation          |
| **Power BI** | KPI visuals, dashboard design, interactive reporting |
| **Python**   | EDA, visualization, trend and correlation analysis   |


###  **Key Insights**

* **Technology** category drives the highest sales and profit
* **West** region performs best in overall revenue
* Higher **discounts** lead to reduced profits
* Sales peak during **November–December** (holiday season)
* Profitability varies significantly across **sub-categories**


###  **Project Deliverables**

*  Excel analysis file ("Superstore.xlsx")
*  SQL queries file (`supersql.sql`)
*  Power BI dashboard (`supply_dashboard.pbix`)
*  Python notebook (`superpython.ipynb`)
